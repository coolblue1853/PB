# PB
 
공부를 위한 개인 프로젝트

<br/><br/>
    
[주제]  
코기 엔진에 던전앤 파이터 전투 방식과, 로프 액션을 추가하여 로그라이트 류 게임을 제작해 보는 것이 목표였습니다.
 <br/>





<br/><br/>
[기술 스택]  

Unity3D / C# / Git 


  


<br/><br/>
[개발 기능]  

로프 액션 시스템 :
기존 엔진에 없던 시스템으로, 화살표 방향으로 로프를 발사, 지정된 TAG와 일치하면 로프가 아래로 생성됩니다.
로프는 방향키를 이용하여 반동을 줄 수 있습니다.
